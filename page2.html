Projetos

Projeto 1: Sistema de Registro de Produção

Desenvolvi um sistema de registro de produção usando VBA Excel. O sistema permite aos usuários registrar informações sobre a produção de itens, incluindo quantidade, data e status.

Projeto 2: TimeLine VBA. Sistema de Planejamento e Controle de Produção

Criei um sistema de planejamento e controle de produção usando VBA Excel. O sistema permite aos usuários planejar e controlar a produção de itens, incluindo informações sobre datas, quantidades planejadas e status.

Projeto 3: Sistema de Análise de Dados com as Sete Ferramentas da Qualidade

Desenvolvi um sistema de análise de dados de produção usando VBA Excel. O sistema permite aos usuários analisar os dados de produção utilizando as Sete Ferramentas Da Qualidade, incluindo informações sobre quantidades produzidas, tempos de produção e eficiência.
